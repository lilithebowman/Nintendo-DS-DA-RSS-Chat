ShoutPro 1.5.2 - What's New?

ShoutPro 1.5 has dozens of new features and bug fixes. An incomplete list is below:

  • View All and User Panel now display inside the main window.
  • New themes allow you to change the look of your shoutbox by replacing one file.
  • Passwords for the User Panel are now encrypted, making ShoutPro more secure than ever.
  • Password entry is now done in the main window rather than in a new JavaScript pop-up, simplifying the shout posting process.
  • A time offset can now be set in config.php to make the timestamp display in your local time rather than the server time.
  • Version checking is now included in the administration panel.
  • Clicking on smilies in the help window now automatically inserts them into your shout.
  • ShoutPro is now released under the Creative Commons Attribution-ShareAlike 2.5 License.

Changes from version 1.5.0 to 1.5.1:

  • Bug Fix: No more JavaScript errors in Internet Explorer
  • Bug Fix: Names with uppercase characters can now properly change or reset passwords

Changes from version 1.5.1 to 1.5.2:

  • Bug Fix: Even less JavaScript errors
  • Bug Fix: Password field does not improperly display at times
