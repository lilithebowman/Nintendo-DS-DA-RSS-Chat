ShoutPro 1.5.2 - Upgrading

Upgrading from Version 1.5.1:
Only one file has been changed from 1.5.1 to 1.5.2. Simply replace your existing shoutbox.php with the new versions.

Upgrading from Version 1.5.0:
Only three files have been changed from 1.5.0 to 1.5.2. Simply replace your existing shoutbox.php, userpanel/dochange.php, and userpanel/dofind.php with the new versions.

Upgrading from Versions Below 1.5.0:
Due to the many changes to the ShoutPro code, ShoutPro 1.5 is intended for new installs only. You can keep your current shouts by backing up "shouts.php" and the files in the "lists" directory, then performing a fresh install and replacing the shouts and list files in your new installation. Please note that the formats of the shoutcode.php and names.php files have changed, and are not backwards compatible. See the customization page for details on converting your old shoutcode.php. All User Panel and restricted names must be re-registered through the User Panel so their passwords will be encrypted.

config.php has also been heavily modified, so please look through the new config.php and manually replace the settings from your old one. If you had customized ShoutPro's graphical display, you will have to make these changes again in a theme CSS file.
